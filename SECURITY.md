# Security Policy

## Supported Versions

Relay is currently in active development. Security updates are provided only for the latest major release.

| Version | Supported          |
| ------- | ------------------ |
| 3.0.x   | :white_check_mark: |
| < 3.0   | :x:                |

## Reporting a Vulnerability

Security is a top priority for Relay. If you discover a vulnerability, we would like to know about it so we can take steps to address it as quickly as possible.

**Please do not report security vulnerabilities through public GitHub issues, discussions, or pull requests.**

Instead, please report them directly via email to:
📧 **manasraj850@gmail.com**

### What to include in your report:
* A detailed description of the vulnerability.
* Steps to reproduce the issue.
* Any potential impact or risk assessment.
* (Optional) Suggested mitigation or patch.

### What to expect:
* You will receive an acknowledgment of your report within 48 hours.
* We will investigate the issue and provide an estimated timeline for a fix.
* Once the vulnerability is resolved, we will publish a security advisory and credit you for the discovery (unless you prefer to remain anonymous).

Thank you for helping keep Relay secure!
